So Another SMB1 Remake?

I wanted to experiment with a paper craft artstyle but I didn't have a project that I could use it on, so I decided to do the good old reliable and remake smb1. A cool fun fact is that I made this game in a week (and it shows haha), but I am working on a rewrite even with an editor.

Screenshots

The Papercraft Artsyle Explained.

Like I said before I'm still working on the game on the side and refining the artstyle the smb1 game is just a testbed for it. Before it was just sprites with a shader that I added a normal and ambient occlusion map to but the way I'm doing it now is that I'm actually generating 3D sprites meshes from the original sprites to cature the paper craft style more.



How the shader is built

The base shader

I essentially use the sprites of the original game and then add normal maps and some ambient occlusion to give them more depth then I add a dynamic paper shader on top of that to give it the cool looking paper craft style.

So whats next for SMB1

I've been updating the game every now and then bolstering it up with brand new features such as: A theme fitting main menu, A cool level selector (with an interactable 3D NES cartidge), and a level editor that lets you build your own levels in the papercraft style! Originally I wanted to streamline the level making process for myself by making a simple level editor but when I finally made one I realised I could refine it and put it into the game for others to make their own levels in the games style.


While I was making the level selector I was playing around with some 3d modelling becuase I wanted the menu to look stylish and interesting so I modelled a cartidge to serve as the menu buttons but I also realised that I could use this cartirdge model to be a display that shows the level you selected on the front like cover art (the feature isn't ready yet but I'm not finished!).
