Autobahn
Raser Reborn

Autobahn Raser is an old German racing series that I tried remaking inside Unreal Engine.

Why did I remake a random old German Game?

Straight up just nostagia, when I was a kid I was at my grandma's house she had a computer with a lot of cds from Computer Spiele. Something I liked a lot about Autobahn Raser was the vibe, it would always feel very emersive to me as I cruised around in the time trial mode. Soo.. years later I had a bit of time on my hands and I tried playing it, sadly though the game was so old that you couldn't play on modern systems without jumping through a bunch of hoops.

Sooo.. I did what any normal person would do....

Rip the meshes and try to recreate the game inside Unreal Engine of course!

Currently the games development is on indefinite hold because I was asked to help with some other development work so I don't have as much time to work on the project as I did although I haven't been doing nothing! Previously I ripped all the meshes on runtime using Ninja Ripper and similar tools.

But now I have been working on reverse engineering the old files of all Davilix Racing games from around the 2000s era, you can see my progress here inside this Gitea Repo. I have exported a bunch of meshes and levels from various games.

Maybe I will try again with the remake but this time with the real meshes?

Why Unreal Engine?

Probably because unreal has the best graphics™... nah I'm just most comfortable right now using unreal engine, I also did not really use the unreal graphics™ as I'm using a pixel shader to make it look more retro. Take a look at the Preview Video!
